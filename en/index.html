This file documents DataCite.

Overview

Classes

Alternate identifier

IRI https://model.ccmm.cz/vocabulary/datacite#AlternateIdentifier
Label Alternate identifier
Definition An identifier other than the primary Identifier applied to the resource being registered. This may be any alphanumeric string which is unique within its domain of issue. May be used for local identifiers, a serial number of an instrument or an inventory number. The AlternateIdentifier should be an additional identifier for the same instance of the resource (i.e., same location, same file).

Contributor

IRI https://model.ccmm.cz/vocabulary/datacite#Contributor
Label Contributor
Definition he institution or person responsible for collecting, managing, distributing, or otherwise contributing to the development of the resource. To supply multiple contributors, repeat this property. For software, if there is an alternate entity that “holds, archives, publishes, prints, distributes, releases, issues, or produces” the code, use the contributorType “HostingInstitution” for the code repository. For instruments, if there is an institution responsible for the management of the instrument (for example, the legal owner, the operator, or an institute providing access to the instrument), use the contributorType “HostingInstitution” for the owner of the instrument.

Creator

IRI https://model.ccmm.cz/vocabulary/datacite#Creator
Label Creator
Definition The main researchers involved in producing the data, or the authors of the publication, in priority order. For instruments this is the manufacturer or developer of the instrument. To supply multiple creators, repeat this property.

Date

IRI https://model.ccmm.cz/vocabulary/datacite#Date
Label Date
Definition Different dates relevant to the work.

Description

IRI https://model.ccmm.cz/vocabulary/datacite#Description
Label Description
Definition All additional information that does not fit in any of the other categories. May be used for technical information or detailed information associated with a scientific instrument.

Description type

Definition The type of the description.

Funder identifier

Funding reference

IRI https://model.ccmm.cz/vocabulary/datacite#FundingReference

Geo location box

IRI https://model.ccmm.cz/vocabulary/datacite#GeoLocationBox

Geo location point

IRI https://model.ccmm.cz/vocabulary/datacite#GeoLocationPoint

Geo location polygon

IRI https://model.ccmm.cz/vocabulary/datacite#GeoLocationPolygon

Definition A drawn polygon area, defined by a set of points and lines connecting the points in a closed chain.

GeoLocation

IRI https://model.ccmm.cz/vocabulary/datacite#Geolocation
Label GeoLocation
Definition Spatial region or named place where the data was gathered or about which the data is focused.

Publisher

IRI https://model.ccmm.cz/vocabulary/datacite#Publisher
Label Publisher
Definition The name of the entity that holds, archives, publishes, prints, distributes, releases, issues, or produces the resource. This property will be used to formulate the citation, so consider the prominence of the role.

Related identifier

IRI https://model.ccmm.cz/vocabulary/datacite#RelatedIdentifier
Label Related identifier
Definition dentifiers of related resources. These must be globally unique identifiers.

Related item

IRI https://model.ccmm.cz/vocabulary/datacite#RelatedItem
Label Related item
Definition Information about a resource related to the one being registered.

Resource

IRI https://model.ccmm.cz/vocabulary/datacite#Resource
Label Resource
Definition Root element of a single record. This wrapper element represents root element taken from XML implementation and is not defined in the DataCite DOI standard. RDF also needs a root element.

Rights

IRI https://model.ccmm.cz/vocabulary/datacite#Rights
Label Rights
Definition Any rights information for this resource. The property may be repeated to record complex rights characteristics.

Subject

IRI https://model.ccmm.cz/vocabulary/datacite#Subject
Label Subject
Definition Subject, keyword, classification code, or key phrase describing the resource.
